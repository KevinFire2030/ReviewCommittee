# 제출용 원페이지 요약

## 1. 발명의 명칭

**온디바이스 인공지능 에이전트를 이용한 자연어 기반 자기 단말 제어 장치 및 방법**

## 2. 발명의 요지

본 발명은 **동일 모바일 단말 내부에서 실행되는 온디바이스 AI 에이전트**가 사용자의 자연어 명령을 해석하고, 무선 ADB 또는 로컬 브리지 기반 제어 경로를 통해 자기 단말의 앱, 운영체제 기능 및 시스템 설정을 직접 제어한 뒤, 실행 결과를 검증하고 실패 시 대체 경로로 재시도하는 **자기 단말 제어 기술**이다.

핵심은 ADB 자체가 아니라 **에이전트의 실행 위치와 제어 주체/대상 관계**이다. 본 발명은 외부 PC, 서버 또는 클라우드 환경이 대상 단말을 제어하는 구조가 아니라, **스마트폰 내부의 AI 에이전트가 동일 스마트폰을 자기 제어하는 온디바이스 아키텍처**이다.

## 3. 청구항 중심 핵심 구성

1. **자연어 명령 수신 및 해석**  
   사용자의 자연어 명령을 수신하고, 수행할 작업과 실행 파라미터를 추출한다.

2. **동일 모바일 단말 내부의 AI 에이전트 실행**  
   AI 에이전트가 외부 PC 또는 서버가 아니라 제어 대상 모바일 단말 내부에서 실행된다.

3. **에이전트 실행 단말과 제어 대상 단말의 동일성 판단**  
   에이전트가 실행되는 단말과 제어 대상 단말이 동일한지 판단하여 자기 단말 제어 구조를 형성한다.

4. **무선 ADB 또는 로컬 브리지 기반 자기 단말 제어**  
   온디바이스 AI 에이전트가 무선 ADB 또는 로컬 브리지를 통해 앱 실행, 화면 조작, 시스템 설정 변경 등 단말 기능을 제어한다.

5. **실행 결과 검증 및 대체 경로 재시도**  
   실행 후 목표 상태 도달 여부를 검증하고, 실패 시 단말 상태와 실패 원인에 따라 다른 실행 경로를 선택하여 재시도한다.

6. **민감 작업 보호**  
   결제, 삭제, 권한 변경 등 민감 작업은 사용자 확인 또는 정책 판단을 거쳐 수행한다.

## 4. 선행기술과의 핵심 차이

| 구분 | 선행기술 조사상 선행기술 | 본 발명 |
|---|---|---|
| 제어 구조 | 외부 제어 환경이 클라우드폰 또는 대상 단말을 제어 | 동일 스마트폰 내부 에이전트가 자기 단말을 제어 |
| 에이전트 실행 위치 | MCP 서버, 컴퓨팅 리소스 호출, 클라우드폰 연결 등 외부/분리 구조로 해석 | 제어 대상 스마트폰 내부에서 온디바이스 실행 |
| ADB의 의미 | 클라우드폰 또는 대상 단말에 연결하기 위한 외부 제어 수단 | 온디바이스 에이전트와 자기 단말 제어 실행부를 잇는 내부 브리지 |
| 제어 대상 | 클라우드폰 또는 외부 대상 단말 | 에이전트가 실행되는 동일 모바일 단말 |
| 기술 효과 | 대상 단말 자동화 또는 원격 제어 | 외부 PC 없는 단말 단독 자연어 제어, 지연/네트워크 의존성 감소, 개인정보 보호, 실행 신뢰성 향상 |

선행기술 조사 보고서에는 `通过ADB连接云手机`, `配置 MCP 服务器`, `调用所述计算资源以执行操作`, `클라우드폰이 ADB를 통해 연결` 등의 표현이 확인된다. 이는 선행기술이 동일 스마트폰 내부의 에이전트가 자기 단말을 제어하는 구조라기보다, 외부 제어 환경 또는 별도 컴퓨팅 리소스가 대상 단말을 제어하는 구조로 해석될 수 있음을 보여준다.

따라서 본 발명과 선행기술의 본질적 차이는 다음과 같다.

> **차이는 ADB가 아니라 에이전트의 실행 위치이다. 선행기술은 외부 제어 환경이 클라우드폰 또는 대상 단말을 제어하는 구조이고, 본 발명은 스마트폰 내부의 온디바이스 AI 에이전트가 동일 단말을 자기 제어하는 구조이다.**

## 5. 기대 효과

- 사용자는 복잡한 스크립트나 메뉴 탐색 없이 자연어로 자기 스마트폰을 제어할 수 있다.
- 외부 PC 또는 서버 의존도를 낮추어 단말 단독 실행 가능성을 높일 수 있다.
- 온디바이스 실행 구조를 통해 지연, 네트워크 의존성 및 개인정보 유출 가능성을 줄일 수 있다.
- 실행 결과 검증 및 대체 경로 재시도 구조를 통해 제어 신뢰성을 높일 수 있다.
- 민감 작업에 대한 사용자 확인 및 정책 판단을 통해 보안성과 안전성을 강화할 수 있다.

## 6. 심의회 제출용 결론

본 발명은 단순한 ADB 자동화 또는 외부 원격 제어 기술이 아니다. 본 발명은 **온디바이스 AI 에이전트가 동일 모바일 단말 내부에서 자연어 해석, 자기 단말 제어, 실행 결과 검증 및 실패 시 대체 경로 재시도를 수행하는 폐루프 자기제어 아키텍처**이다.

따라서 본 발명의 신규성 및 진보성 판단에서는 ADB 사용 여부가 아니라, **에이전트가 제어 대상 단말 내부에서 실행되는지**, **에이전트 실행 단말과 제어 대상 단말이 동일한지**, **그 동일 단말 내부에서 자연어 해석부터 실행 검증까지 수행하는지**를 중심으로 검토할 필요가 있다.
